The realistic target

Automate the repeatable cases, catch the hard cases early.

The goal is not to promise a universal floor-plan parser before seeing BygAI's real distribution. The useful first win is a controlled pipeline that increases throughput while making failure obvious.

Case taxonomy

Separate clean digital plans, scanned plans, low-resolution images, multi-unit pages, missing-scale plans, and visually noisy layouts.

Measurement gates

Define checks for scale, wall continuity, room closure, openings, ceiling assumptions, and quote-relevant surface grouping.

Human review speed

Design the operator checkpoints so the system asks for the smallest correction that unlocks the next automation step.

Implementation slice

End Phase 0 with the next narrow build target, a data contract, and success criteria for production automation.
